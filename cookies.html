Çerez Politikası

Ziravox VR Kulübü web sitesi, kullanıcı deneyiminizi iyileştirmek ve hizmet kalitemizi artırmak için çerezler (cookies) kullanmaktadır. Bu politika, hangi çerezleri kullandığımızı ve nasıl yönettiğimizi açıklar.

  • Çerezler web sitesinin düzgün çalışması için gereklidir.
  • Kullanıcı tercihlerinizi hatırlamak için kullanılır.
  • Site performansını artırmak için analitik veriler toplar.
  • İstediğiniz zaman çerezleri devre dışı bırakabilirsiniz.

Çerez Nedir?

Çerezler, web sitesi tarafından tarayıcınıza gönderilen küçük metin dosyalarıdır. Bu dosyalar:

  • Kullanıcı tercihlerinizi hatırlar
  • Oturum bilgilerinizi saklar
  • Site performansını iyileştirir
  • Kişiselleştirilmiş içerik sunar

Kullandığımız Çerez Türleri

Web sitemizde aşağıdaki çerez türlerini kullanırız:

Zorunlu Çerezler

Bu çerezler web sitesinin temel işlevleri için gereklidir:

  • Oturum yönetimi
  • Güvenlik kontrolleri
  • Dil tercihleri
  • Rezervasyon işlemleri
Analitik Çerezler

Site kullanımını analiz etmek için kullanılır:

  • Ziyaretçi sayısı
  • Popüler sayfalar
  • Kullanıcı davranışları
  • Site performansı
Fonksiyonel Çerezler

Kullanıcı deneyimini iyileştirmek için:

  • Tercih hatırlama
  • Kişiselleştirme
  • Sosyal medya entegrasyonu
  • Form verilerini kaydetme

Üçüncü Taraf Çerezler

Web sitemizde aşağıdaki üçüncü taraf hizmetlerini kullanırız:

  • Google Analytics: Site trafiği analizi
  • Facebook Pixel: Sosyal medya reklamları
  • YouTube: Video içerikleri
  • Google Maps: Konum hizmetleri

Çerez Yönetimi

Çerezlerinizi aşağıdaki yöntemlerle yönetebilirsiniz:

  • Tarayıcı ayarlarından çerezleri devre dışı bırakma
  • Belirli çerezleri silme
  • Çerez uyarılarını etkinleştirme
  • Gizli tarama modu kullanma

Çerez Süreleri

Çerezlerimiz farklı sürelerde saklanır:

  • Oturum çerezleri: Tarayıcı kapatılana kadar
  • Kalıcı çerezler: 1 yıl veya daha uzun
  • Analitik çerezler: 2 yıl
  • Reklam çerezleri: 90 gün

Çerez Tercihlerinizi Değiştirme

Çerez ayarlarınızı değiştirmek için:

  1. Tarayıcınızın ayarlar menüsüne gidin
  2. Gizlilik ve güvenlik bölümünü bulun
  3. Çerez ayarlarını seçin
  4. İstediğiniz seçenekleri etkinleştirin/devre dışı bırakın
